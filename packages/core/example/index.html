Navalone — build-free example

This page loads ../dist/navalone.css and ../dist/navalone.global.js (the minified IIFE) with no build step, proving the <script> workflow still works against the built TypeScript output.

Try it:
  • Hover (or click) the top items to open desktop submenus.
  • Open Products → Developer Tools for a nested flyout.
  • Open Resources for a mega menu with columns.
  • Narrow the window below 960px (or use the button) to collapse into a hamburger + off-canvas drawer.

End of page — scrolling is locked while the mobile drawer is open.
